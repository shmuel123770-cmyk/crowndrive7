CrownDrive - גרסה יציבה עם Netlify Database

מה יש בגרסה:
- לא מחליפה Storage/Database ולא מוסיפה NETLIFY_DB_URL.
- משתמשת ב-Netlify Database דרך @netlify/database/getConnectionString().
- כרטיס רכב כולל צ׳אט עם בעל הרכב.
- הזמנה נוצרת בסטטוס ממתינה לאישור בעל הרכב.
- בעל הרכב יכול לאשר/לדחות.
- הרשמת שוכר מחייבת צד קדמי + צד אחורי של הרישיון + מדינה/State בטקסט חופשי.
- טופס הוספת רכב עם רשימות יצרן/דגם/שנה/צבע/מנוע + אפשרות אחר.
- סוגי השכרה: לפי שעות, ימים, תקופה ארוכה + מחירים מותאמים.
- מיון רכבים לפי פעילות אחרונה updatedAt: שינוי מודעה / הזמנה / צ׳אט מעלים את הרכב למעלה.
- שמירת התחברות באותו דפדפן עד יציאה.
- כפתור הצג/הסתר סיסמה.
- כפתור יציאה מהמערכת.

חשוב לפני העלאה:
1. אל תוסיף NETLIFY_DB_URL אם הוא readonly.
2. אם קיים NETLIFY_DB_URL עם netlifydb_readonly - למחוק אותו.
3. להעלות ל-GitHub ואז לחכות ל-Published ב-Netlify.
4. לבדוק: https://crowndrive770.com/.netlify/functions/db
   צריך להחזיר ok:true אם Database מחובר.
