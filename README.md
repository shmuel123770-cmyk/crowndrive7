# CrownDrive V2
גרסה מודולרית ובטוחה יותר של CrownDrive.

- מאזין Firebase Auth יחיד.
- אין כתיבת state מלאה.
- נתיבים נפרדים למשתמשים, רכבים, הזמנות, הודעות ותשלומים.
- מסמכים פרטיים דרך Signed URLs קצרים לאחר בדיקת הרשאה בשרת.
- כלי העתקת נתונים ישנים שאינו מוחק את המקור.
- בדיקות סטטיות ו-unit tests אמיתיים.
